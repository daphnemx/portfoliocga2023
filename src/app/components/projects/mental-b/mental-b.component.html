Overview

The ADHD Informative Website project aimed to create an educational platform for providing comprehensive information about Attention Deficit Hyperactivity Disorder (ADHD) and its related topics. As the front-end developer for this project, my role was to design and implement a responsive user interface that would ensure a seamless user experience across various devices. I utilized Angular for building the front-end, Firebase for handling user authentication and data storage, and Bootstrap with Sass for responsive and visually appealing styles.

Objective

The primary objective of the ADHD Informative Website was to raise awareness about ADHD, its symptoms, treatment options, and support resources. The platform targeted both individuals seeking information about ADHD and professionals looking for educational content related to the disorder. Additionally, the website featured a diagnostic test to help users assess the probability of having ADHD.

Technologies used
  1. Angular: As the front-end framework, Angular was chosen for its robustness, modularity, and ability to create responsive web applications.
  2. The Firebase platform was used for user authentication, allowing registered users to access exclusive content and upload blog posts securely.
  3. Bootstrap with Sass: The combination of Bootstrap's responsive design components and Sass's CSS preprocessing capabilities facilitated rapid and consistent styling across the website, ensuring responsiveness on all devices.
Features implemented
  1. User Authentication: Users could sign up and log in using their email and password or Google accounts. Authenticated users had access to exclusive content, including blog posts and the ADHD diagnostic test.
  2. Content Management System: Authenticated users were given the ability to create, edit, and delete blog posts through a user-friendly form. Content uploaded by users was stored in Firebase's cloud database, ensuring seamless data management.
  3. ADHD Diagnostic Test: The website featured a multiple-choice test that users could take to assess their likelihood of having ADHD. After completing the test, users received personalized feedback based on their answers, directing them to appropriate resources.
  4. Responsive Design: Bootstrap's responsive grid system and utility classes were utilized to ensure the website adaptability across various devices and screen sizes. The website's layout, fonts, and images automatically adjusted to provide an optimal viewing experience on desktops, tablets, and smartphones.
Challenges faced
  1. Integrating Firebase with Angular for user authentication and data storage required thorough planning and understanding of Firebase's APIs.
  2. Designing a user-friendly interface for the diagnostic test while accurately assessing ADHD probability was a complex task.
  3. Ensuring the website's responsiveness across different devices demanded meticulous testing and optimization.
Conclusion

The ADHD Informative Website project successfully accomplished its objective of creating an educational platform for ADHD-related information. As the front-end developer, I leveraged Angular, Firebase, Bootstrap, and Sass to build a responsive, feature-rich, and user-friendly website. The integration of user authentication, content management, and the diagnostic test provided a valuable resource for users seeking information and support related to ADHD.

The responsive design ensured that the website delivered an optimal experience on various devices, underscoring its accessibility and usability. The project's success highlighted the effectiveness of modern front-end technologies in addressing critical health and educational topics while maintaining a responsive user interface.
